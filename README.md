## 📦 Installation

Clone the repo and use **GNU Stow** to symlink configurations:


git clone https://github.com/moturri/dot.git ~/.dot
cd ~/.dot

yay -S qtile-extras
sudo pacman -S python-psutil brightnessctl acpi xorg-xrandr dunst yazi alsa-utils networkmanager stow
